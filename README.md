# Telegram Codex Bridge website

Static download and setup site for
[Telegram Codex Bridge](https://github.com/adventure-gpt/telegram-codex-bridge).

## Development

```powershell
npm ci
npm run build
npm run preview:cloudflare
```

The production site is deployed to
<https://telegram-codex-bridge.pages.dev/>. It uses no accounts, cookies,
analytics, or server-side data storage.
